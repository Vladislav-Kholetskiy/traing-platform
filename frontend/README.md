# Frontend

Клиентская часть проекта построена на `Vite + React + TypeScript` и предназначена для рабочих экранов платформы обучения.

## Что нужно для запуска

- backend доступен по `http://localhost:8080`
- frontend запускается на `http://localhost:5173`
- установлены Node.js и npm

## Локальный запуск

```powershell
cd frontend
Copy-Item .env.example .env -Force
npm install
npm run dev
```

Стандартный локальный файл `.env`:

```env
VITE_API_BASE_URL=http://localhost:8080
VITE_DEMO_ACTOR_ID=1
```

Переменные:

- `VITE_API_BASE_URL` — адрес backend
- `VITE_DEMO_ACTOR_ID` — идентификатор демонстрационного пользователя для локальной разработки

## Сборка

```powershell
npm run build
```

## Структура приложения

- `src/app/` — каркас приложения, маршрутизация, провайдеры
- `src/features/` — функциональные части по предметным сценариям
- `src/pages/` — страницы верхнего уровня
- `src/shared/` — общие API-утилиты и элементы представления

## Основные разделы интерфейса

- обучающийся: каталог, назначенное обучение, самостоятельное тестирование, результаты, уведомления
- руководитель: текущий контроль и аналитика
- эксперт: управление учебным содержимым и аналитика вопросов
- администратор: пользователи, оргструктура, доступ, назначения, уведомления, импорт, аудит, пересборка аналитики

## Основные обращения к backend

- текущий пользователь: `GET /api/v1/me`
- назначенное обучение: `/api/v1/assigned-learning/assignments`
- самостоятельное тестирование: `/api/v1/self-testing/tests`
- текущие попытки: `/api/v1/current-attempts/...`
- история результатов: `/api/v1/self/results/history`
- управленческая аналитика: `/api/v1/managerial/...`
- управление учебным содержимым: `/api/v1/expert/content/...`
- административные функции: `/api/v1/admin/...`

## Замечания

- состав бокового меню зависит от ролей и доступных разделов, полученных через `GET /api/v1/me`
- `frontend/.env` — локальный файл и не должен попадать в Git
- если описание в этом файле расходится с кодом, ориентироваться нужно на маршруты и реализации в `src/`
