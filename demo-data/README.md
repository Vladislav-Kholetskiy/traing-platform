# Демонстрационные данные

В этой папке собраны версионируемые демонстрационные данные, сценарии запуска и вспомогательные скрипты для повторяемой проверки проекта.

Структура:
- `imports/` — входные наборы данных и базовые SQL-файлы
- `scripts/` — скрипты подготовки, очистки, проверки и быстрой диагностики
- `scenarios/` — пошаговые сценарии, контрольные точки и отчёты о прогонах
- `templates/` — необязательные шаблоны исходных материалов

Правило использования:
- обязательные исходные данные для демонстрации и проверки должны храниться внутри `demo-data`
- путь `tmp/` не считается основным источником обязательных входных данных
- `tmp/` допускается только для временных журналов, локальных снимков и промежуточных артефактов
