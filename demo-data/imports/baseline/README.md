# Базовые SQL-файлы

В этой папке находятся версионируемые SQL-файлы, которые нужны для повторяемой подготовки демонстрационной базы в рамках реализованного сценария.

Файлы:
- `bootstrap-demo-import.sql` — создаёт демонстрационного администратора и его роль для сценариев с заголовком разработчика
- `personnel-demo-baseline.sql` — восстанавливает кадровую и организационную основу, на которой строятся личные, назначенные и управленческие сценарии

Правило использования:
- сценарии запуска и скрипты должны ссылаться на эти файлы, а не на содержимое `tmp/`
- каталог `tmp/` остаётся местом только для локальных и временных артефактов
