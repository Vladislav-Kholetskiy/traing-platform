# Наборы для импорта

В этой папке лежат версионируемые входные данные, которые используются в демонстрационном и приёмочном сценарии.

Содержимое:
- `baseline/` — повторяемые SQL-файлы для начальной подготовки демонстрационной базы
- `org-structure/` — наборы данных по организационной структуре
- `npz-report-content.json` — структурированный набор данных, подготовленный для загрузки учебного содержимого
- `users/` — наборы данных по пользователям и кадровому импорту

Правило хранения:
- обязательные входные данные для описанного сценария запуска должны браться отсюда
- временные файлы из `tmp/` не должны быть обязательной частью документированного процесса

Связанные материалы:
- `demo-data/scripts/extract-npz-report-content.py` — пересобирает `npz-report-content.json` из исходного отчёта
- `demo-data/scripts/import-npz-report-content.ps1` — загружает подготовленный набор данных в работающее приложение
